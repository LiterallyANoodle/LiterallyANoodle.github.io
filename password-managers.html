Password Managers

Why do people use password managers? Why do security professionals recommend password managers? The idea of it is strange to me. After all, you should be remembering your passwords. That's how sites are able to uniquely identify that it is you who is accessing your account - so-called "authentication". The human mind is constantly shifting and moving and losing things in the process. That's not really something you can control, so previously people would resort to using the same password everywhere. This proved to be a bad idea since a bad actor could acquire your password on one site, then use it on another. So then just write it down, of course! Aha but that's another common trap. Many immersive sim games will make use of the ol' password on a sticky note to specifically make their virtual security less secure.

Purportedly, password managers are useful in preventing the forgetting of passwords, the reuse of passwords, and the security of external password storage. And that is true to an extent... But, let's take a step back for a moment and think about what we want to do again with these passwords. We want something that identifies the person who should be allowed to access this account. It's a sort of proof that you are who you say you are. Therefore, we should use something that is unique to the object we call "the person who should be allowed to access this account". On paper, the password solves this by making it so that the correct user is the only object in the entire universe that has this password string in their head (associated with the username of course). (More security-oriented individuals might know this as "Something you know".) Other ways - or perhaps "factors" - of authenticating this user are more things that uniquely identify this object from other objects in the universe.

We've already established that human users are rather flimsy as reliable password storage. So over time, a password may fail to uniquely identify the user from other objects in the universe. It may fail to identify any object in the universe for that matter!
